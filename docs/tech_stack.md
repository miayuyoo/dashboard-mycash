# Documentação da Stack Tecnológica

Este documento lista as principais tecnologias e ferramentas selecionadas para o desenvolvimento do projeto MyCash+, juntamente com seus links oficiais de documentação.

## Frontend & Estilização

### [Tailwind CSS](https://tailwindcss.com/)
Framework CSS utilitário para construção rápida de interfaces customizadas.
- **Uso no projeto:** Estilização global, sistema de design, responsividade e temas.
- **Documentação:** https://tailwindcss.com/docs

### [React Native](https://reactnative.dev/)
Framework para criação de aplicativos nativos para Android e iOS usando React.
- **Uso no projeto:** *Nota: O escopo atual foca na versão Web (Vite), mas esta tecnologia está listada na stack oficial, possivelmente para futura versão mobile nativa.*
- **Documentação:** https://reactnative.dev/docs/getting-started

### [Vite](https://vite.dev/)
Build tool moderna que proporciona um ambiente de desenvolvimento extremamente rápido.
- **Uso no projeto:** Bundler, servidor de desenvolvimento e build de produção para a aplicação Web.
- **Documentação:** https://vite.dev/guide/

## Backend & Infraestrutura

### [Supabase](https://supabase.com/docs)
Alternativa open source ao Firebase. Fornece banco de dados Postgres, Autenticação, APIs instantâneas e Realtime.
- **Uso no projeto:** Banco de dados (PostgreSQL), Autenticação de usuários, Armazenamento de arquivos e APIs de dados.
- **Documentação:** https://supabase.com/docs
